# Jornada Python - Hashtag Treinamentos

Este repositório contém os projetos e exercícios práticos desenvolvidos durante a **Jornada Python**, organizada pela Hashtag Treinamentos. O objetivo é aplicar conceitos de Python em situações reais do mercado, como análise de dados, automação e inteligência artificial.

## 📂 Estrutura do Repositório

O projeto está dividido em quatro módulos principais:

### 1. 📊 Análise de Dados
Focado no tratamento de bases de dados para identificação de padrões e tomada de decisão.
* `inicial.ipynb`: Notebook com a análise exploratória.
* `cancelamentos.csv`: Base de dados utilizada para o estudo de churn.

### 2. 📦 Cadastro de Produtos
Automação de processos repetitivos utilizando bibliotecas de controle de periféricos (como PyAutoGUI).
* `codigo.py`: Script principal da automação.
* `produtos.csv`: Lista de dados para preenchimento automatizado.

### 3. 🤖 Chatbot com IA
Desenvolvimento de uma interface de chat integrada com modelos de linguagem (OpenAI).
* `main.py`: Código principal do chatbot.
* `Alternativo/`: Versões secundárias e gabaritos com templates HTML/CSS.

### 4. 🧠 Machine Learning
Aplicação de modelos preditivos para análise de crédito ou comportamento do cliente.
* `inicial.ipynb`: Treinamento e teste dos modelos.
* `clientes.csv` / `novos_clientes.csv`: Dados históricos e novos dados para previsão.

---

## 🛠️ Tecnologias Utilizadas
* **Python**
* **Pandas**: Manipulação e análise de dados.
* **PyAutoGUI**: Automação de processos.
* **Scikit-learn**: Implementação de modelos de Machine Learning.
* **OpenAI API**: Integração com inteligência artificial.
